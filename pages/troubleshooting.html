Troubleshooting

Find detailed solutions step-by-step how to fix common VO issues

Check FAQs Get Started

1. Fullscreen Issues

If your fullscreen mode is bugged, it may be caused by your launcher or game settings. Follow these steps to fix the issue:

1. Open Options → Video Settings → Fullscreen, and set it to Off.
2. Turn On Fullscreen again.
3. If it still doesn't work, try to change the resolution of the game.
4. If it still doesn't work, try to change the resolution of your monitor.
5. If it still doesn't work, try Fullscreen Off & bordless window and then Fullscreen On.
6. If it still doesn't work, try to uncheck the FullScreen option in your launch settings/version settings.
7. If it still doesn't work, try to reinstall the game and/or the modpack.

2. FPS Issues

If you're experiencing lag despite having a high-end computer, try the following steps to improve performance:

1. Open Options → Video Settings → Fullscreen, and set it to Off.
2. Turn On Fullscreen again. - This will make the game faster and more stable.
3. Lower your Render Distance/Distance Simulation/Max Shadow Distance to 12 or lower.
4. Set your Graphics to Fast instead of Fancy.
5. If using shaders, switch to a lower-quality preset (Potato, Very Low, Low, or Medium) or disable shaders entirely.
6. Toggle VSync On/Off to see if it improves performance.
7. Allocate more RAM to the game in your launcher settings.
8. If the issue persists, try reinstalling the modpack.
9. Some Minecraft versions are poorly optimized, which might be the root cause of performance issues. Consider trying a different version.
