Getting Started

Get started with Velocita Optimized

Check Troubleshooting Check Performance Tips

1. Downloading the modpack

First, before you can start playing Velocita Optimized, you need to pick a hosting platform. We recommend using Modrinth to download the modpack.



1. Open Modrinth and search for Velocita Optimized.
2. Click on the versions button.
3. Pick a modpack version. ++ Versions have vanilla QoL & versions without it is pure optimization.
4. Read the version description.
5. Download the version.

2. Setting Up

After downloading the modpack let's setup the modpack



1. If you downloaded directly in the launcher, you're good to skip to step 7
2. Make sure you know where the modpack version has saved into your file explorer.
3. Open your launcher.
4. Go to version manager(this is on sklauncher).
5. Import the file you saved/downloaded before.
6. Select the version you imported.
7. Go to instance configuration.
8. Set the memory to 2GB or more.
9. Configuration done!

3. Playing the game

After downloading & setting up the modpack let's play!



1. Play the instance
2. Wait for the game to load
3. Done! You're now playing Velocita Optimized!
4. If you have any issues, check the troubleshooting page.
5. If you have any questions that weren't answered in FAQs, join our discord server.
6. If you found an issue that wasn't reported, report it in the issue tracker or join our discord server.
7. Enjoy the game!
